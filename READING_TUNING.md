# Phase 17 読取精度強化メモ

今回の強化点:
- 帳票タイプ自動判定
  - receipt
  - invoice
  - parking_receipt
  - medical_receipt
  - credit_slip
  - purchase_screenshot
- OCR後の正規化強化
  - 日付 YY/MM/DD → YYYY-MM-DD
  - 店名や説明のノイズ除去
  - 明細の広告・QR・会員文言除去
- 支払方法の推定強化
  - 現金
  - クレジットカード
  - コード決済
  - 電子マネー
  - iD
- カテゴリ推定の辞書拡張
  - ガソリン
  - 駐車場
  - ホームセンター
  - 衣料
  - 保険系
  - 医療系
- プロンプト強化
  - 駐車券は「入庫 / 精算 / 現金 / 500円」優先
  - 医療系は「金額 / 発行日 / 医療機関名」優先
  - スクショは「商品代金 / 購入日時 / 商品ID / 配送方法」優先

次に精度をさらに上げる候補:
- 画像前処理の追加
  - 自動トリミング
  - 傾き補正
  - コントラスト強化
  - 感熱紙向け二値化
- 帳票タイプごとのレビューUI分岐
- 重複検知に documentType を加味
